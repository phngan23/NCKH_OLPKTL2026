# Predicting Modelling Impulsive Buying Behavior on TikTok Shop Livestream: An Integrated SEM and Machine Learning Approach
* The code is the official implementation of the work "Predicting Modelling Impulsive Buying Behavior on TikTok Shop Livestream: An Integrated SEM and Machine Learning Approach"
* Authors of the work: Pham Van Huy, Nguyen Phuong Ngan, Do Pham Ha Chi, Dinh Nguyen Anh Thu, Nguyen Dang Quynh Tien.
* Supervisor of the work: Dr. Vu Thi Bich Ngoc

### Idea
The idea of planned behavior, a psychological framework, and machine learning have all been combined in this study to produce models that accurately predict consumer purchasing behavior. The inclusion of XAI also explores the most significant influencing factors that drive online purchasing.
